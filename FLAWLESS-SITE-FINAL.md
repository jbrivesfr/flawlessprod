## DESIGN NOTES FOR DEVELOPER

- **Primary CTA everywhere:** WhatsApp deep link (like Esther's iMessage button) — opens chat with pre-filled "Hi Lucy!" message
- **Mobile-first** — majority of users will land from phone
- **No login required** — emphasize this. No app, no account.
- **Agent cards on homepage** — Lucy and Luna side by side, clear personality, photo/avatar
- **Testimonial section** — use card format with name + title (not just quotes)
- **"How It Works" section** — 3 steps, icon + number + short description (steal Esther's structure)
- **Sticky nav CTA** — "WhatsApp Lucy" button always visible on scroll
- **Color:** Keep Flawless brand, but CTA buttons should use WhatsApp green (#25D366) to reinforce the interface
- **FAQ:** Accordion format. Keep it scannable.
- **Footer:** Minimal. Logo, tagline, links, copyright.

## TONE GUIDE (FOR FUTURE COPY)

**Voice:** Direct. Warm. Confident. No corporate buzzwords.

**Do:**
- "Lucy fixes that" (declarative)
- "Stop doing X" (problem callout)
- "Here's what's broken" (diagnosis)
- Specific examples over vague promises
- "You" language throughout

**Don't:**
- "Transform your workflow" (buzzword)
- "Leverage synergies" (corporate)
- "Helps you optimize" (hedge)
- "Best-in-class" (meaningless)
- Passive voice
